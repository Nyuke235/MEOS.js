MEOS.jsTemporal data types for JavaScript

TypeScript/WebAssembly bindings for MobilityDB MEOS, available in the browser and Node.js.

Get Started
API Reference

WASM-powered

Runs the full MobilityDB MEOS C library in WebAssembly, no server required, works in any modern runtime.

Strongly typed

Every type is a TypeScript class with TSDoc annotations, full IDE autocompletion, and explicit memory management.

Rich type system

Integers, floats, dates, timestamps, each with span, span-set, and set variants, plus temporal bounding boxes.
