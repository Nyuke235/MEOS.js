meos.js / TInterpolation

Enumeration: TInterpolation ​

Defined in: types/temporal/Temporal.ts:44

Interpolation mode of a temporal sequence.

MEOS supports four modes: None -> used for Instant subtypes (no interpolation concept) Discrete -> values are defined only at the listed instants Stepwise -> value is constant between instants (step function) Linear -> value is linearly interpolated between instants (TFloat only)
